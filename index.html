KLMS / Canvas API 同期

Canvas Access Tokenを入力して、履修講義・曜日時限・教室・課題・提出状況を同期します。
Keio IDのパスワードは入力しないでください。入力したトークンはこのアプリには保存されず、同期リクエスト中だけ使用されます。
この機能はindex.htmlの直接起動では使えません。必ずnpm startで起動してください。

時間割はCanvasのTimetable、講義カレンダー、講義名・セクション、シラバス本文の順で曜日・時限を補完します。 KLMS側に曜日・時限が登録されていない講義は「未設定」として講義一覧に残します。 

アクセストークンを入力して「トークンを確認」を押してください。

JSON貼り付けインポート

APIトークンが発行できない場合は、KLMSにログインしたブラウザで https://lms.keio.jp/api/v1/planner/items を開き、表示されたJSONをここに貼り付けて取り込めます。
